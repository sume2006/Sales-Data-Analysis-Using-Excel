# Sales-Data-Analysis-Using-Excel
Project Overview

This project presents a comprehensive Sales Performance Dashboard built using Microsoft Excel. The dashboard analyzes sales data using Pivot Tables, KPI metrics, and visualizations to generate meaningful business insights.

The objective of this project is to demonstrate practical data analysis skills using Excel.

Tools & Techniques Used
Microsoft Excel
Pivot Tables
KPI Metrics
Data Cleaning
Data Aggregation
Data Visualization
Business Analysis
Key Performance Indicators (KPIs)
Total Sales
Total Profit
Total Quantity Sold
Profit Margin (%)

Profit Margin Formula Used:
Profit Margin = Total Profit ÷ Total Sales

Dashboard Insights

The dashboard provides analysis on:

Sales by Region
Sales & Profit by Category
Top Performing Products
Customer Segment Analysis

These insights help evaluate overall business performance and profitability.

Project Structure
Data Sheet – Contains raw sales dataset
Pivot Table Sheet – Aggregated analysis using Pivot Tables
Dashboard Sheet – Final dashboard with KPIs and charts
Business Value

This dashboard helps in:

Monitoring overall sales performance
Identifying high-performing regions and products
Evaluating profit margins
Supporting data-driven decision making
Skills Demonstrated
Data Cleaning

KPI Calculation
Dashboard Design
